# Expert System – Simple Disease Diagnoser

## Description
This project implements a rule-based expert system for disease diagnosis.

## Features
- Loads diseases and symptoms from `diseases.txt`
- Lists all diseases
- Diagnoses diseases by asking yes/no questions
- Eliminates impossible diseases based on user responses
- Identifies the most probable disease

## Files
- `expert_system.py` – Main program
- `diseases.txt` – Knowledge base containing diseases and symptoms

## Concepts Used
- Expert Systems
- Knowledge Base
- Inference Engine
- Rule-Based Reasoning
- Dictionaries
- Sets
- File Handling
